# Competitive Benchmark

## References

| Product / Pattern | Strength | Gap This Project Covers |
| --- | --- | --- |
| Stream Deck | 物理ボタン操作と状態表示が強い | Codex pet表示とM5Stack実機運用へ寄せる |
| Home Assistant Dashboard | LAN内deviceと通知連携が強い | Codex確認待ちへの短い返信に特化する |
| Desktop notifier | PC通知が軽い | PC画面外の常設pet端末にする |
| MQTT Dash / Blynk | IoT端末連携が軽い | Codex Appのpet変更と回答表示を扱う |

## Differentiation

- Codex Appのpetと通知を同じ端末で扱う。
- Yes/No/Otherや3択返信を机上端末で完結させる。
- Core2とGRAYの両方に対応し、タッチ非搭載deviceにも操作代替を用意する。
