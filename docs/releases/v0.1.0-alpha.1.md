# v0.1.0-alpha.1

## 位置づけ

`m5stack-codex-pet-notifier` の closed alpha prerelease です。Host Bridge mock、JSON event contract、Core2 / GRAY device profile、simulator、sample telemetry、firmware scaffold、release docs を同梱します。

## 検証結果

- `cmd.exe /d /s /c npm test`: pass
- `docs/platform-runtime-gate.json`: simulator / mock device / sample telemetry / device adapter / host adapter / security/privacy pass
- `dist/validation-result.json`: representative suite pass
- QCDS: Quality `S-`、Cost `A+`、Delivery `A+`、Satisfaction `S-`

## 手動テスト準備

- Core2 と GRAY を同一 Wi-Fi に接続できる環境を用意する。
- Host Bridge は LAN 内に bind する。
- pairing token を発行し、token なし device event が拒否されることを確認する。
- 実機操作の記録は `docs/manual-test.md` の表に追記する。

## 手動テスト未実施

手動テストはCodexでは未実施です。Core2 / GRAY の firmware build、flash、Wi-Fi 再接続、touch、swipe、button、IMU tap、実 Codex App adapter 連携は未確認です。
